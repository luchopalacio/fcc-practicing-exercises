# Exercise: Build a recursive countdown

This function generates a countdown array starting from a given number down to 1 using recursion.

## Rules
-Uses recursion only (no loops)
-No array iteration methods
-Returns a new array on each call

## Example

```js 
countdown(5); // Output: [5, 4, 3, 2, 1]

countdown(1); // Output: [1]

countdown(0); // Output: []
